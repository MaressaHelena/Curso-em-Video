## Operadores Aritméticos

### adição
```py
+ # usado para somar

# exemplo: 
print(5+2)
```

### subtração
```py
- # usado para subtrair

# exemplo: 
print(5-2)
```

### multiplicação
```py
* # usado para multiplicar

# exemplo: 
print(5*2)
```

### divisão
```py
/ # usado para dividir

# exemplo: 
print(5/2)
```

### potência
```py
** # usado para elevar um x qualquer a um termo ( x²)

# exemplo: 
print(5**2)
```

### divisão inteira
```py
// # retorna o número inteiro de uma divisão

# exemplo: 
print(5//2)
```

### resto da divisão
```py
% # é literalmente o resto de uma divisão

# exemplo: 
print(5%2)
```
